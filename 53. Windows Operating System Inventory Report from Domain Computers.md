# 🖥️ Windows Operating System Inventory Report from Domain Computers

## 📌 Overview

This documentation explains how to generate a **Windows Operating System version inventory report** from **Domain Computers** using **PowerShell**.

The report will collect operating system information from computers joined to the domain and produce a **CSV report showing:**

* Computer Name
* Operating System Version
* Total number of devices running each OS version

This procedure is useful for **IT asset inventory, upgrade planning, and security compliance checks**.

# ⚙️ PowerShell Script for OS Inventory Report

The following PowerShell script collects the **Operating System attribute from Active Directory computers**, groups them by OS version, and exports the results to a CSV report.

```powershell
Get-ADComputer -Filter * -Property OperatingSystem |
Where-Object {$_.OperatingSystem -ne $null} |
Group-Object OperatingSystem |
ForEach-Object {
    $Count = $_.Count
    $OS = $_.Name
    $_.Group | Select-Object Name,
    @{Name="OperatingSystem";Expression={$OS}},
    @{Name="TotalCountForThisOS";Expression={$Count}}
} |
Sort-Object OperatingSystem |
Export-Csv "C:\Windows_Versions_With_Count.csv" -NoTypeInformation -Encoding UTF8
```

# 🚀 How to Run the Script

⚠️ **Important:**
This script must be executed **only on the Active Directory Server**.

### Step 1 — Open PowerShell

1. Open **PowerShell**
2. Right-click **Windows PowerShell**
3. Select **Run as Administrator**

---

### Step 2 — Execute the Script

Copy and run the following script:

```powershell

Where-Object {$_.OperatingSystem -ne $null} |
Group-Object OperatingSystem |
ForEach-Object {
    $Count = $_.Count
```
